### 📋 Sobre o Projeto
O **Glow & Style** é uma aplicação de agendamento e vitrine de serviços desenvolvida para salões de beleza e centros de estética. O objetivo principal é proporcionar uma jornada de agendamento visual e intuitiva, valorizando o portfólio do profissional e facilitando a reserva de horários pelos clientes.

Desenvolvido para ser rápido, elegante e focado na conversão de novos agendamentos.

---

### 🚀 Stack Técnica
Utilizei tecnologias que permitem uma interface rica em detalhes e um carregamento instantâneo:

- **Frontend:** React + TypeScript.
- **Estilização:** Tailwind CSS (Layout fluido e adaptável).
- **Componentes:** Shadcn/UI (Consistência visual).
- **Ferramentas:** Vite + Bun (Ambiente de alta performance).

---

### ✨ Funcionalidades Principais
* **Vitrine de Serviços:** Exibição detalhada de cortes, colorações e tratamentos.
- **Agendamento Inteligente:** Interface preparada para integração com calendários.
- **Totalmente Responsivo:** Experiência de uso perfeita em tablets e smartphones.
- **SEO Ready:** Estrutura otimizada para buscadores.
